# 🌐 Personal Portfolio Website

## 📌 Overview

This is a modern, fully responsive personal portfolio website developed to showcase my skills, projects, and experience as a Software Engineering student. The website focuses on clean UI design, smooth animations, and an engaging user experience, following modern web development practices.

---

## 🚀 Features

* 🎨 Elegant dark-themed UI with aqua gradient accents
* 📱 Fully responsive across all devices
* ✨ Smooth animations and transitions
* 🌌 Interactive particle background
* 🧠 Skills section with animated progress bars
* 📂 Projects section with structured card layout
* 📩 Contact section with user-friendly interface
* ⚡ Optimized performance and clean code structure

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Bootstrap
* **Scripting:** JavaScript
* **Design Concepts:** Flexbox, CSS Grid, Animations

### 1. Clone the Repository

```

```

```

```

### 3. Run the Project

Simply open `index.html` in your browser.
