# 🌐 Personal Portfolio Website

A responsive and modern personal portfolio website built using HTML and Bootstrap. This website showcases projects, skills, and contact information in a clean, professional layout suitable for freelancers, developers, or job seekers.

---

## 📌 Overview

Your portfolio website acts as your **online identity**. It highlights your skills, projects, and experience in a visually appealing and accessible way. This project emphasizes responsive design, usability, and clear presentation of personal and professional information.

---

## 🚀 Features

* 🌐 Fully responsive design for desktop, tablet, and mobile
* 🎨 Clean UI using Bootstrap components
